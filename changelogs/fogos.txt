HertzifyOS Android 16 for Motorola Moto G34/45

* Vanilla & Gapps available
* MotoCamera & Dolby are included
* Rooted with ReSUkiSU
* Signed build

How to install:
==============================
# VANILLA #
* Boot to TWRP/OFOX -> Wipe data -> Flash ROM.zip -> (Flash TWRP/OFOX to ramdisk if you want to flash Gapps) Reboot Recovery -> Flash Gapps.zip -> Factory reset (format data) -> Reboot system
* reflash boot.img if you wanted ReSukiSU root

# GAPPS #
* Boot to TWRP/OFOX -> Wipe data -> Flash ROM.zip -> Reboot recovery -> Format data -> reboot system
